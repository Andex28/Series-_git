De que trata el primer capitulo

Veamos los primeros pasos con HTML5 y Visual Studio Code, que es lo que necesitamos para comenzar a trabajar con HTML y CSS desde cero y paso a paso.En este capítulo del curso de desarrollo web te mostraré Visual Studio Code, un editor de código que nos hace la vida muy fácil con diversos lenguajes de programación.Además aprenderás a escribir tus primeras líneas de código HTML y CSS paso a paso, para que la práctica haga al maestro!

¿Que es el css?

CSS, en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.
