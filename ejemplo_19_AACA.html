Navegación lateral (SideNav) fija con altura máxima

Intente desplazarse por esta área y observe cómo el navegador lateral se adhiere a la página.

Observe que este elemento div tiene un margen izquierdo del 25%. Esto se debe a que la navegación lateral está configurada en un 25% de ancho. Si elimina el margen, el navegador se superpondrá o se ubicará encima de este div.

Observe también que hemos configurado overflow:auto en sidenav. Esto agregará una barra de desplazamiento cuando el navegador lateral sea demasiado largo (por ejemplo, si tiene más de 50 enlaces dentro).
